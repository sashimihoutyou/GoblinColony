- [ ] **A1. 汎用ジョブキュー + 建築システム**（Godot / §5・§7・spec 3-12/3-15）

  採掘（`RESOURCE_NODE`→`EXHAUSTED`）、部屋建築フロー（建築モード→ゴースト→2タップ確定→

  **前提: A1a. 資源スカラー 6 種**（food/wood/mud/herb/equipment/gems / spec 3-11）。
  §10 の資源/建築/撤去ダイヤルはこの上に乗る（→B7）。
- [ ] **A2. ブリーチング + 防衛配分 UI**（Godot / §9・P2-07・P3-06）
  壁破壊敵 + 破壊予告（`breach_warnings` + 警告ハイライト）、破壊されない壁、
  防衛ライン（DefensePoint = 巣口内側 2 タイル）+ 配分スライダー + 敵戦力比例の自動配分、
  襲撃フェーズ進行（予兆→装備取得→配分→交戦→復興。現状は予兆 ETA と自動陣形のみ）。
  `WALL_HP=20` は暫定値（§NUMBERS）。
- [ ] **A3. 中立ルート状態機械 + 4 ルート選択**（§14.5.7・P3-08）

  （UI でなく状態で出口を弾く）、難度設定画面（開始時 易/並/難 + 外交 4 ルート + 規定日数）。
  TS の humanHostility 無ドリフト設計（PR #19）が土台。
- [ ] **A4. 外征の完全実装**（§11.5・P3-02）
  出現物の残り 5 系（動物/旅人/ゴブリン放浪者=外部加入/敵性キャンプ/廃墟 + 行き倒れの少女フック）、
  距離属性（近い/遠い）、戦闘系外征（出発前に泥鍛冶屋で装備→人数で成否→返り討ちは負傷止まり）、
  難度目安表示、業の漏れ（§13 接続）、遅れて来る増援の防衛ライン自動合流（現状は即時解除の P1 簡易版）。
  外交資源（宝石・毛皮 →B5）と外部加入の供給源を兼ねる合流点。
